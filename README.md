# Student Course Management System

## Overview

Student Course Management System is a Java-based console application designed to manage students, teachers, courses, and enrollments.

This project is being developed step by step using proper software engineering practices including requirement analysis, design, implementation, testing, documentation, deployment, and future AI integration.

## Current Version

Version 1: Console-based Java OOP application

## Main Objectives

- Learn Java OOP from basic to advanced level
- Build a clean and modular project structure
- Follow industry-standard software development lifecycle
- Practice documentation and testing
- Prepare the project for future database, backend, frontend, AI, Docker, and CI/CD integration

## Current Features Planned

- Add student
- View all students
- Search student by ID
- Update student
- Delete student
- Add teacher
- View all teachers
- Add course
- View all courses
- Enroll student in course
- View enrollments
